## 📝 Описание релиза

Первый полнофункциональный релиз бота Отсчёт с поддержкой всех основных функций для создания и управления отсчётами до важных событий.

### 🎯 Основные достижения v1.0.0

✅ **Полностью функциональный бот**
- Создание отсчётов с поддержкой 4 шаблонов
- Управление отсчётами (просмотр, удаление)
- Интерактивный календарь для выбора дат

✅ **Система уведомлений**
- Персональное время уведомлений для каждого пользователя
- Ежедневные напоминания через APScheduler
- Поддержка ежегодных повторений

✅ **Архитектура**
- Асинхронный код на Aiogram 3
- Типизированный Python с type hints
- Структурированная работа с БД через SQLAlchemy
- Миграции через Alembic

✅ **Документация**
- Полный README с инструкциями
- Примеры использования
- API документация
- Руководство для разработчиков

## 🚀 Установка

### Быстрый старт

```bash
# 1. Клонирование
git clone https://github.com/Mukller/countdown-bot.git
cd countdown-bot

# 2. Виртуальное окружение
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows

# 3. Установка зависимостей
pip install -r requirements.txt

# 4. Конфигурация
cp .env.example .env
# отредактируйте .env с вашим BOT_TOKEN и DATABASE_URL

# 5. Миграции БД
alembic upgrade head

# 6. Запуск
python -m app.main
```
