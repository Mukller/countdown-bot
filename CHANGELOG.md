# Changelog

Все важные изменения в этом проекте будут задокументированы в этом файле.

## [1.0.0] - 2025-05-16

### 🎉 Добавлено
- Первый релиз Отсчёт
- Основной функционал создания отсчётов до событий
- Система ежедневных уведомлений с персональным временем
- Интерактивный календарь для выбора дат
- Поддержка повторяющихся ежегодных событий
- Четыре шаблона событий (День рождения, Отпуск, Экзамен)
- Сохранение всех отсчётов в базе данных PostgreSQL
- Красивый интерфейс с инлайн-кнопками
- Логирование всех действий
- Поддержка русского языка

### ✨ Особенности
- Асинхронная архитектура на Aiogram 3
- SQLAlchemy ORM для работы с БД
- APScheduler для отправки ежедневных напоминаний
- FSM (Finite State Machine) для управления состояниями
- Middleware для внедрения зависимостей (AsyncSession)
- Graceful shutdown с отменой всех задач

### 📦 Зависимости
- aiogram==3.0.0
- SQLAlchemy==2.0.0
- asyncpg==0.28.0
- python-dotenv==1.0.0
- apscheduler==3.10.0
- alembic==1.12.0
- pydantic-settings==2.0.0
